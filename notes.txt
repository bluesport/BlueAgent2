Don’t forget:
	cd web_code
	npm install

To start server:
	node_modules/.bin/webpack-dev-server

Alternatively (if you’ve configured package.json):
	npm start

If you encounter an error, first try npm install. If you still get the error:
	losf -w -n -i tcp:8080
	kill -9 [problematic PID]

Alternatively:
	killall -9 node

If you get any npm problems, try these:
	cd web_code
	npm install
	npm install react-infinite-grid
